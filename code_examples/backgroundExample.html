Background-image method

- This example uses background-image property .


You can use this construct to style images that are difficult to make responsive.

Specify fixed dimensions for the wrapper and set the path to the image using the property background-image

background-size: cover property sets the size of the element's background image. The image can be left to its natural size, stretched, or constrained to fit the available space.
