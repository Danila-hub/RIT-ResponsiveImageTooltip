Picture TAG method

- Use html picture tag for your images, which will allow you to select an image for each display resolution.

Use the srcset parameter to specify the address for your image.
The media parameter specifies the resolution at which the selected image will be displayed.
Img is a tag for the default image that is displayed by default in IE browser or if the display resolution does not support other images.
